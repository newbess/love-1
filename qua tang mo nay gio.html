Trend code trái tim là gì? Cách làm code trái tim của thủ khoa Lý cực dễ

(VOH) - Mấy ngày nay, code trái tim của thủ khoa Lý đang hot rần rần trong giới trẻ. Làm sao để “đu” được trend này, hãy cùng VOH tìm hiểu nhé!
    Mục lục
    1. 1.Code trái tim là trend gì của gen Z?
    2. 2.Nguồn gốc của trend làm code trái tim
    3. 3.Làm sao để “đu” trend code trái tim của thủ khoa Lý? Cách tạo code trái tim đơn giản

    Trên nền tảng “tóp tóp” đang lan truyền nhanh chóng các đoạn clip làm code trái tim để tỏ tình crush. Trào lưu này nhanh chóng được người trẻ yêu thích và tìm cách để “đu” theo. Nếu bạn là một trong số họ thì hãy đọc kỹ bài viết sau đây, chúng tôi sẽ bật mí cho bạn cách thực hiện trend này nhanh chóng mà chỉ cần vài các bước đơn giản.

    1. Code trái tim là trend gì của gen Z?

    Code trái tim là một đoạn một ngôn ngữ lập trình mà coder đưa vào máy tính để mã hóa nó thành hình trái tim chuyển động cực kỳ đặc sắc. Vì thành phẩm được tạo ra rất đẹp và lãng mạn nên trend này đã được các bạn trẻ gen Z rất yêu thích. Nhanh chóng những đoạn video tạo code trái tim với hình dạng, màu sắc khác nhau đã trở nên phổ biến, hot rần rần trên mạng xã hội và được nhiều người tìm cách làm theo.

    2. Nguồn gốc của trend làm code trái tim

    Bạn có thắc mắc là trend làm code trái tim được bắt nguồn từ đâu không? Thực ra nó được phổ biến là nhờ vào bộ phim “Thắp sáng anh, sưởi ấm em” của Trung Quốc đang được các bạn trẻ cực kỳ yêu thích. Cụ thể trong tập 5 của phim, nhân vật Lý Tuân hay còn gọi là Thủ khoa Lý do Trần Phi Vũ thủ vai đã vẽ nên một trái tim màu hồng bằng code cực kỳ đặc biệt để tặng cho crush của anh là Chu Vận (Trương Tịnh Nghi). 

    Ngay lập tức phân cảnh ngọt lịm này đã được netizen xứ Trung bắt trend nhiệt tình, các đoạn clip bắt chước hoặc biến tấu trong việc tạo code trái tim nhanh chóng được lan truyền rộng rãi trên mạng. Chính nhờ vậy mà trong vài hôm gần đây nó cũng gây bão trên nền tảng facebook và tiktok tại Việt Nam. Dưới độ hot thế nên ngoài phiên bản gốc, code trái tim còn có rất nhiều phiên bản độc đáo, ấn tượng được sáng tạo từ cộng đồng mạng như code tim đỏ, code tim có gắn thêm chữ,...

    3. Làm sao để “đu” trend code trái tim của thủ khoa Lý? Cách tạo code trái tim đơn giản

    Nếu đang muốn tán đổ crush, hoặc đơn giản chỉ là "đu" trend tạo code trái tim của thủ khoa Lý thì bạn không nên bỏ qua phần này nhé! Chúng tôi sẽ hướng dẫn bạn các bước thực hiện vô cùng đơn giản dưới đây. Để làm được chiếc trend thú vị này chúng ta cần làm theo 3 bước sau:

    Bước 1: Ở hệ điều hành windows, bấm tổ hợp phím Windows + S sau đó gõ Notepad để tìm và mở trình soạn thảo Notepad.

    Bước 2: Kế đến bạn hãy copy một trong hai đoạn code dưới đây vào file.

    Code trái tim của Thủ khoa Lý

    Code-trai-tim-cua-Ly-Tuan.txt

    Code trái tim đập có dòng chữ I LOVE YOU

    Code-trai-tim-dap-voi-dong-chu-i-love-you.txt

    Bước 3: Bấm Ctrl + S nhập tên tùy chọn theo cú pháp ten-file.html ví dụ Qua-tang-ban.html, sau đó mở file bạn sẽ thấy hiệu ứng code trái tim tuyệt vời này.

    Bước 4: Gửi file cho người yêu, crush qua email hoặc ứng dụng nhắn tin như messenger, zalo và nhắc người ấy mở trên máy tính để thấy món quà của bạn.

    Lưu ý:

    Hiện nay tình trạng đánh cắp dữ liệu qua mạng rất phổ biến, các hacker có thể chèn mã độc vào các file code trái tim vì thế bạn không nên click vào các đường dẫn, file của người lạ để tránh bị đánh cắp dữ liệu nhé!

    Vừa rồi chúng tôi đã giới thiệu xong một chiếc trend “code trái tim” siêu lãng mạn gửi đến quý đọc giả. Để công cuộc tán crush trở nên dễ dàng hơn bạn có thể thử vẽ lên một trái tim ngọt ngào, dễ thương dành tặng họ nhé. Chúc bạn may mắn!
